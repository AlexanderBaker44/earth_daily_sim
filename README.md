* Make sure to download Jupyter Notebook to run the simulation with the given inputs
* https://www.anaconda.com/download/
* Once anaconda prompt is downloaded open the Anaconda prompt and run pip install notebook
* From anaconda prompt run the command Jupyter Notebook
* Open the file earth_daily_sim_run.ipynb
* The inputs can be changed by updating the sat_params for the constellation and the scen_params for the scenarios
* After the parameters are updated, the notebook can be run and the new results will be saved within the data folder
* The constellation delays, the scenario delays and the delays per satellite
